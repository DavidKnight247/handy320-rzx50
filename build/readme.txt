You need the lynx bios to run (not included in the opk).

Place in the $HOME/.handy directory.

Atari Lynx roms can be zipped or unzipped.

Controls:

D-Pad: Movement
A/B: A/B (swappable from menu)
X/Y: Option 1 & 2
L:Menu
R:Change aspect ratio
SELECT:quit

Changelog:
Doublebuffering enabled
Hardware (IPU) scaling enabled
Menu reconfiguration, removed unused options.
